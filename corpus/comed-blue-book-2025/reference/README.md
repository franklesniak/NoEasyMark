## Purpose

This document contains ComEd's official guide for installing or revising
electric service. It is a key reference for:

- Planning new or upgraded electric service installations
- Understanding ComEd's service connection process and requirements
- Coordinating with ComEd for residential service work
- Complementing the Service and Meter Requirements with practical
  installation guidance
