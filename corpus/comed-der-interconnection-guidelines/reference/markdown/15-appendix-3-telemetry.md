# Appendix 3: Telemetry

## Communication Options for Projects greater than 5 MW

| Type | Benefits | Risks | Estimated Costs | Timing |
| --- | --- | --- | --- | --- |
| Fiber – ComEd installed | Satisfies PJM participation which requires 2 second polling rate | Higher costs; Fiber installation process will take long time up to year round or more; Single spur is less reliable | · Included in Study | Per ComEd construction schedule |
| Wireless RADWIN | ComEd generally has infrastructure available; High bandwidth low latency | Separate pole required at Interconnection; Customer location; Site survey required to determine availability | Provided after detailed engineering; Estimated rough cost per link is \$80,000 – 90,000; Estimated rough cost for installing a pole is \$120,000 - 150,000 | Per ComEd construction schedule |
| Fiber Ethernet (Carrier) – 3rd Party install | Typically lower cost than ComEd installation due to potential closer connection point | 3rd party carriers don't offer/guarantee SLAs for availability; Higher costs are also tied to locked in contracts; Dedicated fiber installation process will take long time per vendor; Monthly fees; Single spur limits redundancy options | · 3rd party estimates costs · Install costs · Estimated rough monthly cost: \$500-1,500 · Ongoing lease costs · Router & Firewall Install · Router & Firewall O&M | 3rd party construction schedule |
| Wireless Radio Network (MAS, SSN) | Not an option due to 5-minute pole cycle | | | |
| Jet Stream – public Internet with firewall behind it - direct to PJM | Satisfies the need for limited real-time telemetry, but not for revenue meter sharing | Limited number of projects allowed to be installed; Only allows limited operational data; Data would come to ComEd SCADA through ICCP link; Requires carrier ethernet connection; Back to back firewall required; Monthly fees; Need PJM to provide limited data | · 3rd party estimates · Install costs · Ongoing lease costs · Router & Firewall Install · Router & Firewall O&M | 3rd party construction schedule |
